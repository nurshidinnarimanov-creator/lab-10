# lab-10
Калькулятор процентов на JavaScript

Простое веб-приложение, демонстрирующее принципы модульности и структуры проекта с использованием ES-модулей.
Программа позволяет вычислить процент от числа и сумму числа с прибавленным процентом.

Функционал

Ввод исходного числа и процента.

Расчёт:

процента от числа;

числа с прибавленным процентом.

Мгновенное обновление результата.

Кнопка сброса значений.

Современный адаптивный дизайн с плавной анимацией и градиентными кнопками.

Используемые технологии

HTML — структура страницы;

CSS — визуальное оформление и адаптивность;

JavaScript (ES Modules) — логика приложения и модульная архитектура:

mathUtils.js — математические функции;

formatter.js — форматирование результата;

main.js — главный модуль, связывающий всё вместе.

Структура проекта
lab10/
├── index.html
├── style.css
├── main.js
├── mathUtils.js

Как использовать

Открой проект в браузере (или через локальный сервер).
Введите исходное число в поле «Число».
Укажите процент в поле «Процент».
Нажмите кнопку «Рассчитать».
Результат появится под формой.
Чтобы сбросить значения, нажмите «Сброс».

Цель лабораторной работы

Изучить принципы модульности программ.

Научиться использовать import / export в JavaScript.

Разделить логику проекта на независимые части.

Создать современный интерфейс с помощью CSS.

Результат работы

После запуска пользователь получает удобный и стильный калькулятор процентов, демонстрирующий применение модульной структуры и современного интерфейса.
